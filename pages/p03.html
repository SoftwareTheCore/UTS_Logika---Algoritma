Struktur Kontrol - Percabangan (Branching)

üõ£Ô∏è Apa itu Percabangan?

Percabangan adalah struktur yang memungkinkan program memilih jalur berdasarkan kondisi tertentu. Seperti di persimpangan jalan - pilih kiri atau kanan?

Analogi Sederhana:
"Jika hujan, bawa payung. Jika tidak hujan, tidak perlu payung."
Ini adalah percabangan dalam kehidupan sehari-hari!

1Ô∏è‚É£ IF Sederhana

Hanya mengecek satu kondisi, jika True maka jalankan perintah

Syntax:

if kondisi:
    pernyataan
Contoh:

nilai = 85
if nilai >= 70:
    print("Selamat! Anda LULUS")

# Output: Selamat! Anda LULUS

2Ô∏è‚É£ IF-ELSE

Ada 2 pilihan: jika kondisi True jalankan A, jika False jalankan B

Syntax:

if kondisi:
    pernyataan_jika_true
else:
    pernyataan_jika_false
Contoh 1: Cek Genap/Ganjil

angka = 7
if angka % 2 == 0:
    print("GENAP")
else:
    print("GANJIL")

# Output: GANJIL
Contoh 2: Cek Kelulusan

nilai = 65
if nilai >= 70:
    print("LULUS")
else:
    print("TIDAK LULUS")

# Output: TIDAK LULUS

3Ô∏è‚É£ IF-ELIF-ELSE

Ada BANYAK pilihan kondisi. Cek dari atas ke bawah, yang pertama True akan dijalankan

Syntax:

if kondisi1:
    pernyataan1
elif kondisi2:
    pernyataan2
elif kondisi3:
    pernyataan3
else:
    pernyataan_terakhir
Contoh: Konversi Nilai ke Grade

nilai = 78

if nilai >= 80:
    grade = "A"
elif nilai >= 70:
    grade = "B"
elif nilai >= 60:
    grade = "C"
elif nilai >= 50:
    grade = "D"
else:
    grade = "E"

print(f"Grade Anda: {grade}")
# Output: Grade Anda: B

4Ô∏è‚É£ Nested IF (IF Bersarang)

IF di dalam IF - untuk kondisi yang lebih kompleks dan bertingkat

Contoh: Cek Kelulusan dengan Syarat Ganda

nilai = 85
absen = 80

if nilai >= 70:
    if absen >= 75:
        print("LULUS")
    else:
        print("TIDAK LULUS - Absen kurang")
else:
    print("TIDAK LULUS - Nilai kurang")

# Output: LULUS

üîó Kombinasi dengan Operator Logika

Lebih simpel menggunakan and, or, not

Dengan AND:

nilai = 85
absen = 80

if nilai >= 70 and absen >= 75:
    print("LULUS")
else:
    print("TIDAK LULUS")

# Lebih simpel dari nested if!
Dengan OR:

hari = "Sabtu"

if hari == "Sabtu" or hari == "Minggu":
    print("Weekend - Libur!")
else:
    print("Hari kerja")

üé≤ Flowchart Percabangan

Dalam flowchart, percabangan digambarkan dengan belah ketupat (‚óá)

  • Dari belah ketupat keluar 2 garis: Ya dan Tidak
  • Jalur Ya menuju ke blok jika kondisi benar
  • Jalur Tidak menuju ke blok jika kondisi salah
üí° Tips Penting untuk UTS:
  • IF-ELIF-ELSE dicek dari ATAS ke BAWAH
  • Yang pertama True akan dijalankan, sisanya diabaikan
  • ELSE tidak wajib, tapi IF-ELIF harus ada kondisi
  • Indentasi (spasi di awal) SANGAT PENTING di Python!
  • Untuk kondisi ganda, pakai and/or lebih baik dari nested if

üß™ Latihan Trace
