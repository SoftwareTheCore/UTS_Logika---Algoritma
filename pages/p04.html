Struktur Kontrol - Perulangan (Looping)

üîÅ Apa itu Perulangan?

Perulangan (loop) adalah struktur yang mengulang perintah berkali-kali sampai kondisi tertentu terpenuhi.

Analogi:
Bayangkan Anda diminta menulis "Saya akan belajar" sebanyak 100 kali.
Tanpa loop: tulis 100 kali manual üò´
Dengan loop: suruh komputer ulangi 100 kali otomatis! üòä

üî¢ Jenis Perulangan di Python

1Ô∏è‚É£ FOR LOOP

Digunakan ketika jumlah perulangan sudah diketahui/pasti

Syntax Dasar:

for variabel in range(n):
    pernyataan

üìù Memahami range()

  • range(5) ‚Üí 0, 1, 2, 3, 4 (5 angka, mulai dari 0)
  • range(1, 6) ‚Üí 1, 2, 3, 4, 5 (dari 1 sampai 5)
  • range(1, 10, 2) ‚Üí 1, 3, 5, 7, 9 (loncat 2)
  • range(10, 0, -1) ‚Üí 10, 9, 8, ..., 1 (mundur)
Contoh 1: Cetak 1 sampai 5

for i in range(1, 6):
    print(i)

# Output:
# 1
# 2
# 3
# 4
# 5
Contoh 2: Hitung Total 1+2+3+4+5

total = 0
for i in range(1, 6):
    total = total + i
print(f"Total: {total}")

# Output: Total: 15
Contoh 3: Tabel Perkalian

angka = 5
for i in range(1, 11):
    print(f"{angka} x {i} = {angka * i}")

# Output:
# 5 x 1 = 5
# 5 x 2 = 10
# ...
# 5 x 10 = 50

2Ô∏è‚É£ WHILE LOOP

Digunakan ketika jumlah perulangan belum diketahui, tergantung kondisi

Syntax:

while kondisi:
    pernyataan
    update_variabel # PENTING! Jangan lupa!
Contoh 1: Cetak 1 sampai 5

i = 1
while i <= 5:
    print(i)
    i = i + 1 # atau i += 1

# Output: 1 2 3 4 5
Contoh 2: Input sampai benar

password = ""
while password != "1234":
    password = input("Masukkan password: ")

print("Password benar! Akses diberikan")
‚ö†Ô∏è BAHAYA: Infinite Loop!

# JANGAN LAKUKAN INI!
i = 1
while i <= 5:
    print(i)
    # Lupa i += 1, loop tidak pernah berhenti!
Solusi: Pastikan ada statement yang membuat kondisi jadi False!

üõë BREAK dan CONTINUE

BREAK - Keluar dari loop paksa

for i in range(1, 11):
    if i == 5:
        break # Berhenti saat i = 5
    print(i)

# Output: 1 2 3 4 (berhenti sebelum 5)

CONTINUE - Lompat ke iterasi berikutnya

for i in range(1, 6):
    if i == 3:
        continue # Lewati angka 3
    print(i)

# Output: 1 2 4 5 (3 dilewati)

üîÑüîÑ Nested Loop (Loop Bersarang)

Loop di dalam loop - untuk pola 2 dimensi atau tabel

Contoh 1: Pola Bintang

for i in range(3):
    for j in range(5):
        print("*", end="")
    print() # Pindah baris

# Output:
# *****
# *****
# *****
Contoh 2: Pola Segitiga

for i in range(1, 6):
    for j in range(i):
        print("*", end="")
    print()

# Output:
# *
# **
# ***
# ****
# *****
Contoh 3: Tabel Perkalian Lengkap

for i in range(1, 4):
    for j in range(1, 4):
        print(f"{i} x {j} = {i*j}")
    print("---")

üéØ Contoh Soal Umum

Soal: Hitung faktorial dari n
Faktorial 5! = 5 √ó 4 √ó 3 √ó 2 √ó 1 = 120

n = 5
faktorial = 1

for i in range(1, n+1):
    faktorial = faktorial * i

print(f"{n}! = {faktorial}")
# Output: 5! = 120
Soal: Cek bilangan prima

angka = 7
prima = True

for i in range(2, angka):
    if angka % i == 0:
        prima = False
        break

if prima:
    print(f"{angka} adalah bilangan prima")
else:
    print(f"{angka} bukan bilangan prima")
üí° Tips Penting untuk UTS:
  • range(n) dimulai dari 0 sampai n-1
  • range(1, n+1) untuk 1 sampai n
  • WHILE harus ada kondisi yang membuat loop berhenti
  • break keluar dari loop, continue lompat iterasi
  • Nested loop: loop luar berjalan 1x, loop dalam berjalan penuh
  • print(..., end="") untuk cetak tanpa pindah baris
