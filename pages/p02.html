ğŸ“¦ Apa itu Variabel?

Variabel adalah tempat menyimpan data di memori komputer. Bayangkan seperti kotak yang bisa diisi dan diberi label.

Contoh Variabel:

umur = 20 # Variabel umur berisi 20
tinggi = 175.5 # Variabel tinggi berisi 175.5
adalah_mahasiswa = True # Variabel boolean

ğŸ¨ Tipe Data di Python

1. Integer (Bilangan Bulat)

Angka tanpa koma, bisa positif atau negatif

2. Float (Bilangan Desimal)

Angka dengan koma (titik desimal)

Kumpulan karakter yang diapit tanda petik (' atau ")

4. Boolean

Nilai True (benar) atau False (salah)

sudah_menikah = False
aktif = True

ğŸ”„ Konversi Tipe Data (Type Casting)

Mengubah dari satu tipe ke tipe lainnya:

  • int() - Ubah ke integer
  • float() - Ubah ke float
  • str() - Ubah ke string
Contoh:

angka = "100" # String
angka_int = int(angka) # Jadi 100 (integer)

nilai = 85 # Integer
nilai_str = str(nilai) # Jadi "85" (string)
